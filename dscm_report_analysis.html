Overview

This script processes the dscm annotation dataset (provided by DARC) for inclusion in expedition report.
Data has been prepped using 01_data_prep.qmd and 02_refined_taxa_list.R.

In prepping the data, we refine our taxonomic richness metrics, by refining our interpretation of what to count as a distinct taxon.
Without this refinement, we would just be counting how many scientificName entries are fish or inverts.
But we want to exclude higher-level taxonomic IDs if a lower-level taxon is present, e.g. we don’t count both Macrouridae and Coryphaenoides longicirrhus for a species richness of 2.

Here’s a step to QA/QC the logic.
In the process of refining our data, review the table below to see exactly what taxa the code decided to exclude.
This is a list of every single higher-level taxon that was successfully dropped (like Macrouridae, or Ophiuroidea) because a lower-level ID bumped it out.
